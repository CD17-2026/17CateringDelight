# 17CateringDelight

# 🍽️ 17 Catering Delight

A modern and responsive catering service website developed as an assignment for the **Internet Technology & Multimedia** course.

The website allows customers to explore catering services, browse menu packages, request quotations, and book catering events through a simple and user-friendly interface.

**Live Website:** https://17cateringdelight.online/

---

# 📖 About the Project

17 Catering Delight is a catering service website developed to provide customers with an efficient online platform for discovering catering services and making event bookings.

The website features a clean interface, responsive design, and interactive elements to ensure a smooth browsing experience across different devices.

---

# 🎯 Objectives

- Provide complete business information.
- Promote catering services and menu packages.
- Allow customers to book events online.
- Enable customers to request quotations.
- Create a responsive and user-friendly website.
